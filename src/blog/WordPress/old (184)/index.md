- 外判給兩公司做場景  
    
      
    2. 3D 掃描現實遺址

    - 模組會動就是靠這個系統

- 所有角色使用統一骨架  
    
      
    - 新增角色動作時不需要再為每一個角色獨立創造
      
    - 新技能可以使用更多非預設動作
      
    - 新技能可以使角色更加靈活
