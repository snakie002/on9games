## <u> 裝備選擇 </u>

守護者下限保證 傷害目標為
技能等級 -> 天賦 -> 星團
再下一步就是每個裝備欄位都需要召喚物傷害加持
大後期部份 前期專心盡量撐高等級及星團即可

其他散件主要為血抗裝即可保證紅圖

頭：血抗頭 +召喚物等級為佳
身：有護甲有能盾為佳 5連優先
手：護甲血抗
鞋：護甲血抗 ＋ 移速
武器：+1全法術 / 召喚 / 火焰都可湊著用 後綴上觸發插糟技能即可
盾：血抗能盾 +1 召喚早期較奢侈 能有則好

![](post_assets/E1.png)  
項鍊：惡咒護符

戒指：三圍 > 血抗 > 召喚物傷害 > 召喚物攻速
皮帶：夜惡降臨

## <u> 感想心得 </u>
